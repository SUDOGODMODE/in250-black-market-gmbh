# Sicherheitsrichtlinie der Black Market Software GmbH

## Unterstützte Versionen
Sicherheitspatches werden aktuell nur für die Version v1.0 (Release) bereitgestellt.

## Schwachstellen melden
Sollten Sie eine Sicherheitslücke entdecken, melden Sie diese bitte nicht öffentlich über ein Issue. 
Senden Sie stattdessen eine verschlüsselte Nachricht an das Hauptquartier von Lord Black (oder kontaktieren Sie den Repository-Inhaber privat).

Wir bemühen uns, alle Meldungen innerhalb von 48 Stunden zu prüfen.
